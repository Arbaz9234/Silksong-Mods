## Build Steps

Clone the repository:

```bash
git clone https://github.com/Arbaz9234/Silksong-Mods.git
cd silksong-mods
```

Open the `.sln` file in Visual Studio 2022 and build the project.

Compiled DLLs will be generated in:

```text
bin/Debug/
```

If you are new to Silksong modding, start with this setup/tutorial guide:

👉 https://www.youtube.com/watch?v=QMiWbu0il6w
