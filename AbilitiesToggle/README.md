# AbilitiesToggle

A lightweight mod for Hollow Knight: Silksong that allows enabling or disabling abilities at any time during gameplay.

## Overview

Hornet's movement at the start of the game can feel extremely limited.  
This mod allows you to freely toggle in-game abilities whenever you want.

## Features

- Toggle any ability at any time
- Keeps track of acquired abilities through game progression
- Use Faydown Cloak without requiring the Drifter's Cloak
