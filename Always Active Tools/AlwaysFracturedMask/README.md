# Always have Fractured Mask

Permanently equips the Fractured Mask tool

## Overview

This mod is for Hollow Knight: Silksong, which makes the Fractured Mask effect permanent on your tools without explicitly equipping or acquiring it.

## Features

- Your tools permanently have Fractured Mask equipped.
- No need to equip the tool, allowing you an extra tool slot for other tools.
- The Fractured Mask's protection will always prevent Hornet's death while it is active.
- Once broken, the Fractured Mask will be repaired at a bench to restore its protection.
