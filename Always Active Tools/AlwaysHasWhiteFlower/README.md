# Everbloom Bypass - Skip Act 3 (White Flower Unlock)

Instantly grant Hornet the Everbloom (White Flower) and protection from the void so you can face Lost Lace and finish Act 3 whenever you like.

## [Spoiler warning]

This mod alters story gating. It lets you bypass Old Hearts, Elegy of the Deep sequences, and other Act 3 progression. Only install if you accept skipping major plot beats and trophies/achievements may be affected.

The game will always treat Hornet as if she has recovered the Everbloom from the Red Memory.

## Features

- Toggleable at any time - enable or disable the Everbloom effect whenever you want without restarting the game
- Hornet is treated as having the Everbloom / White Flower while the mod is enabled
- Allows Hornet to dive into the Void.

## How to use

- Keep it enabled after your first Abyss visit to finish the game immediately.
- Keep it disabled to play Act 3 normally (bosses, memories, progression)
- Enable it again anytime you are ready to return to the Abyss and finish the game early.
