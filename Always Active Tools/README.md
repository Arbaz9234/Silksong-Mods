# Always Active Tools

Collection of lightweight Silksong mods that keep specific tools permanently active without requiring a tool slot.

## Included Mods

- AlwaysFracturedMask

Each mod is contained in its own folder with a dedicated README explaining functionality, setup, and configuration.

---

# Purpose

These mods are intended to:

- Remove the need to equip certain utility tools
- Keep passive/QoL effects permanently enabled
- Simplify tool loadouts during gameplay
